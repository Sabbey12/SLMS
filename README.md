# Student Learning Management System (SLMS)

## Description
SLMS is a web-based system that allows lecturers to manage courses and learning materials while students can access course content, submit assignments, and view grades online.

---

## Features

### Student
- Login
- View course materials
- Submit assignments
- View grades

### Lecturer
- Upload learning materials
- Create assignments
- Grade student submissions

---

## Technology Used
- HTML
- CSS
- PHP
- MySQL
