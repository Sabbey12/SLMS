## Project Description

This project is a Student Learning Management System (SLMS) Course Profile Module developed for CSEB5223 Software Construction & Methods.

The system allows users to manually input course profile data, search, edit, delete, and display course information using a console-based interface.

## Features

- Add new course profile
- Prevent duplicate course codes
- Search course using linear search
- Edit course information (course code cannot be edited)
- Delete course with confirmation message
- View all course profiles
- Automatic display after Search, Edit, and Delete operations
- Input validation and error handling

## Class Design

Course Class Attributes:

- Course Name
- Course Code
- Credit Hour
- Course Summary
- MS Teams Link

## Software Construction Practices

- Modular method design (single responsibility functions)
- Proper naming conventions
- Input validation implemented
- Linear search algorithm used
- Confirmation before deletion
- Documentation header comments added
